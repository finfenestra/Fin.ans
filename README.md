# FIN.ANS – Tax & Compliance Calculator Portal

FIN.ANS is a responsive web-based Indian tax utility platform built using HTML, CSS, and JavaScript.

## Features
- Income Tax Calculator (FY 2019-20 onwards)
- Old vs New Regime Comparison
- Section 89 Relief Calculator
- Form 10E Draft Working
- Official Government Resource Links
- Responsive Futuristic UI
- Glassmorphism Design
- Print/PDF Export Support

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Purpose
Designed for tax professionals, payroll processing, compliance assistance, and individual taxpayers seeking quick tax estimation tools.

## Disclaimer
This project is intended for educational and utility purposes only. Always verify tax computations with official government portals and qualified professionals.
